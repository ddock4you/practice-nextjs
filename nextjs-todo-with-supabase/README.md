# NEXT.JS Todo List

## 프로젝트 폴더 구조

```
root/
├── actions/      # 서버 액션 관련 코드
├── app/          # 페이지 라우팅 및 레이아웃
├── components/   # 재사용 가능한 UI 컴포넌트
├── config/       # 환경 설정 파일
├── containers/   # 상태 관리 및 비즈니스 로직
├── public/       # 정적 파일 (이미지, 폰트 등)
└── utils/        # 유틸리티 함수
    └── supabase/ # Supabase 클라이언트 설정
```
